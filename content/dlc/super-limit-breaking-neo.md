**Super Limit-Breaking NEO** — the **Season 2 / "DLC 4"** paid expansion. **ANNOUNCED but NOT YET RELEASED** as of **2026-06-10**. Launch window is **"Summer 2026"** (detailed at DRAGON BALL Games Battle Hour 2026, **Apr 19, 2026**). The `releaseDate` here is a **placeholder** (2026-07-01) pending an official date — treat as unconfirmed. the price is **not announced**: Bandai has **not** announced a price (community speculation ~$30+; research/06 §Gaps).

> The name nods to the PS2-era Japanese naming (Sparking! → **NEO** → METEOR). Bandai has **not** officially branded it "Season Pass 2."

## What it adds

| Aspect | Detail |
|---|---|
| **New characters** | **30+** ("over 30" marketing); **27 catalogued** so far (roster slots **#209–235**). **DP values unpublished** — not yet in any DP tier list. |
| **New stages** | **4** — confirmed: **Kami's Palace**, **Stratosphere (Planet Vegeta)** (+2 more). |
| **New solo mode** | **Limit Breaker Journey** — RPG-lite branching scenarios → fights/events → level-up; growth carries into online; 20+ customization items. |
| **New mechanics** | **Chain Blast** and **Sparking! Boost** (shipped via free update — see [[ranked-match]] / mechanics). |
| **Customizations** | **20+** costumes & Super Attacks, incl. a new Goku (Z-End) costume + Super Attack. |
| **Theme song** | **"ZERO"** by **Hironobu Kageyama** (first DB Sparking theme from him in ~18 years). |
